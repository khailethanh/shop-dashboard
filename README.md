# MyShop Dashboard

A personal dashboard for tracking and understanding my own Etsy shop —
orders, fulfillment, shop performance, and customer reviews in one
place.

## What this is (and isn't)

This is a personal learning project. It is built by me, for my own
shop, and is not intended for public users or third-party
sellers. It does not collect, store, or share any data with anyone 
other than me.

## What it does

- **Shop overview** — shop name, Star Seller status, vacation mode,
  last sync time
- **Order tracking & fulfillment** — view orders, filter by status
  and date, mark orders as shipped with tracking info, flag overdue
  orders needing attention
- **Analytics** — revenue trends, average order value, orders by day
  of week, conversion rate, top-performing listings by views and
  revenue
- **Review management** — read shop reviews, track ratings, respond
  to reviews (response is stored locally; Etsy's public API does not
  currently support posting review replies)

## Tech stack

Node.js, Express, EJS, Chart.js, SQLite (`better-sqlite3`)
